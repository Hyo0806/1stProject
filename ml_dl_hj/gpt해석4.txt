시간대별 오늘 매출 실시간 예측 구조

[시간대별 과거 데이터]
→ 시간대별 전용 모델 학습
→ 오늘(요일 + 실시간 날씨) 입력
→ 시간대별 오늘 매출 예측

사용하는 입력 변수
DAY : 오늘 요일
TEMP : 실시간 기온 (기상청)
RAIN : 실시간 강수량 (기상청)


모델	시간대 평균 R²
RandomForest	0.60 ~ 0.68
XGBoost	            0.68 ~ 0.78

저녁 시간대(17~22시) 성능 개선 폭 큼
날씨 영향이 큰 날에 안정적

출력
HOUR별 예상 매출(AMT)
